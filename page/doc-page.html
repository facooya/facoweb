Release Note

Release note by v1.2.0

I apologize for missing a few features in the v1.2.0 update. Originally, I had planned to include dark/light mode and a code theme, but unfortunately, they couldn't be implemented in this release. Due to burnout, my work pace has slowed down, and I plan to take some time off to recharge so I can deliver better features in the next version.

Previously, the header and footer were managed together, but starting from v1.2.0 and onward, they will be updated independently. Once these independent updates are ready, I will share the header and footer code as open-source on GitHub.

Before taking a break, I will upgrade the content to align with v1.2.0, and after that, I'll return.
