# Faco Main
Just slide effect for content when open `drawer-menu` on `faco-header`, on Tablet and Desktop screen size.
Not mobile size support, that intend it.
The `faco-main` tag one-time per page.

---

## Quick Start
Link stylesheet and add tag:
```html
<link rel="stylesheet" href="../../ui/faco-main/index.css">
<faco-main></faco-main>
```

> [!TIP]
> Style: `/ui/faco-main/index.css`
