# Data Format
```javascript
const FacoTabData = {
	title: "Title",
	tabs: {
		"Tab label": {
			"Box label": {
				link: "https://.../box/link",
				text: "preview description"
			}
			// box: add from here
		}
		// tab: add from here
	}
};
```

Don't forgot the trailing comma`,`.  
Refer to `./data.js` as a guide.
